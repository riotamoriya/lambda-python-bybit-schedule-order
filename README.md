# bybit スケジューリング成行注文ツール

## 概要

このツールは、取引所bybitのAPIを使用して自分のアカウントに注文を出すものです。注文に関するエラーハンドリングは実装されていないため、使用時には注意が必要です。このツールの主な開発目的は、任意のタイミングで注文を出すことです。主なユースケースは、成り行き注文をスケジューリングすることですが、APIドキュメントに従えば任意の注文発注が可能です。

## 機能

- 発注
- スケジューリング
